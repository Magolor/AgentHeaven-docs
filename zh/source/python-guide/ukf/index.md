# UKF (通用知识格式)

统一知识格式 (Unified Knowledge Format, UKF) 是一个旨在标准化知识表示的框架。它系统地组织来自异构源的信息，例如数据库模式、文档和历史用户查询，以实现企业系统内知识的便捷和自动化长期维护。UKF作为通用的协议，支持多种存储后端（如数据库、文件系统）和多样化的搜索索引，并提供溯源跟踪、版本控制和知识库个性化等附加功能。

当前UKF版本：v1.0

兼容UKF版本：v1.0
