# 知识库引擎 - 分配需求

本文档概述了 AgentHeaven 架构中知识库引擎组件的分配需求。

## 1. 概述

知识库引擎提供对UKF格式存储的知识进行检索和利用的能力。

<br/>

## 2. 分配需求

### 2.1 功能需求

| ID | 需求 | 实现细节 | 优先级 |
|----|------|----------|--------|
| FR-08 | 检索引擎 | 实现搜索方法 | 高 |
| FR-14 | 知识蒸馏 | 支持模型微调 | 低 |
| NFR-01 | 性能 | 优化检索速度 | 高 |
| NFR-02 | 可观测性 | 记录检索操作 | 中 |

<br/>

### 2.2 跨组件依赖

| 组件 | 描述 |
|------|------|
| KLStore | 用于访问存储的知识 |
| LLM | 用于知识增强操作 |
| Agent | 用于利用检索到的知识 |
