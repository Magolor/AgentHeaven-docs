# BaseUKF - 分配需求

本文档概述了 AgentHeaven 架构中 BaseUKF 组件的分配需求。

## 1. 概述

BaseUKF 是 AgentHeaven 中所有知识表示的基础类。

<br/>

## 2. 分配需求

### 2.1 功能需求

| ID | 需求 | 实现细节 | 优先级 |
|----|------|----------|--------|
| FR-01 | 统一知识格式 | 实现 UKF 格式 | 高 |
| FR-07 | 知识表示 | 支持多种知识类型 | 高 |
| NFR-02 | 可扩展性 | 支持自定义知识类型 | 中 |
| NFR-04 | 文档 | 记录 UKF 模式 | 中 |

<br/>

### 2.2 跨组件依赖

| 组件 | 描述 |
|------|------|
| 知识库存储 | 用于持久化 UKF 记录 |
| 知识库引擎 | 用于检索 UKF 记录 |
| 代理 | 用于使用 UKF 记录 |
