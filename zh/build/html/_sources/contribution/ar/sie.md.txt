# 结构化信息提取 (SIE) - 分配需求

本文档概述了 AgentHeaven 架构中结构化信息提取(SIE)组件的分配需求。

## 1. 概述

SIE组件处理从各种数据源中提取结构化信息。

<br/>

## 2. 分配需求

### 2.1 功能需求

| ID | 需求 | 实现细节 | 优先级 |
|----|------|----------|--------|
| FR-06 | 工具管理 | 实现提取工具 | 高 |
| FR-11 | 数据库原生 | 提取数据库模式 | 高 |
| FR-17 | 分布式系统 | 支持分布式提取 | 低 |
| NFR-03 | NL2SQL准确性 | 确保准确的模式提取 | 高 |

<br/>

### 2.2 跨组件依赖

| 组件 | 描述 |
|------|------|
| 数据库 | 用于模式提取 |
| BaseUKF | 用于结构化输出格式 |
| 知识库存储 | 用于存储提取的模式 |
